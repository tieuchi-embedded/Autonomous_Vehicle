I/O Bridge — io/serial_bridge

serial_node
bin/serial_node
Sub MQ CONTROL_CMD
Pub MQ SERIAL
UART /dev/ttyACM0 · 115200 baud · full-duplex
Uplink parse @IMU:yaw,pitch,roll;; · @RPM:rpm;; → pub SERIAL
Downlink poll CONTROL_CMD → write #RPM:val;; #STR:val;;
Safety SIGTERM → gửi rpm=0, steer=0 trước khi thoát

Perception — perception/

camera_sim_node
bin/camera_sim_node <video.mp4> [fps]
Input MP4 file (OpenCV VideoCapture), loop on EOF
Output SHM CAMERA_FRAME — 320×180 BGR
Note Thay bằng camera_node khi có camera thật (libcamera)
state_node
bin/state_node
Sub MQ SERIAL (SerialData)
Pub MQ EGO_STATE (EgoState)
Logic angle = yaw (pass-through)
speed = (rpm / 7) × 2π × 6.5 / 60 cm/s
gear ratio 7:1, wheel radius 6.5 cm
lane_node
bin/lane_node [--show]
Sub SHM CAMERA_FRAME
Pub MQ LANE_STATE
Pipeline 1. IPM warp — bird's-eye view
   src trapezoid: TL=0.15, TR=0.85, top_y=0.30
2. HLS binary — L channel > 200 (white lines)
3. Sliding window — 9 windows, histogram seed
4. Polyfit bậc 2 — x = b·y + c
5. EMA tracker — α=0.7, ưu tiên right lane, fallback left
Output heading_err_rad — góc tiếp tuyến poly vs trục xe
lateral_offset_m — khoảng cách ngang tâm xe vs tâm lane

Control — control/

control_node
bin/control_node
Sub MQ BEHAVIOR_CMD
MQ EGO_STATE
Pub MQ CONTROL_CMD
Rate ~50 Hz (CLOCK_MONOTONIC dt)
Speed PID Controller
Kp=10.0, Ki=0.5, Kd=0.1
input: target_speed (m/s) vs speed_cms
output: rpm ∈ [−300, +300]
Lateral Pure Pursuit (lookahead 0.3 m)
input: target_heading (deg)
output: steer_deg ∈ [−30, +30]
manual_node
bin/manual_node
Pub MQ CONTROL_CMD
Keys w/s — RPM ±step
a/d — steer ±step
space — full stop
x — center steer
q — quit
Config env: MANUAL_RPM_STEP, MANUAL_RPM_MAX,
MANUAL_STEER_STEP, MANUAL_STEER_MAX
